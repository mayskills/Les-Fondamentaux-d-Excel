
# Module 1 – Interface et Navigation

## Objectif pédagogique
À l’issue de ce module, l’apprenant sera capable de :
- Identifier les principaux éléments de l’interface Excel
- Comprendre la structure d’un fichier Excel
- Naviguer efficacement dans une feuille de calcul
- Sélectionner des cellules de manière précise

---

## 1. Concepts fondamentaux

### 1.1 Le classeur
Un classeur est un fichier Excel. Il constitue le support global de travail et peut contenir plusieurs feuilles.  
Exemple : un classeur peut regrouper un budget annuel, des dépenses mensuelles et un suivi des recettes.

### 1.2 Les feuilles
Une feuille est une page de travail à l’intérieur du classeur.  
Chaque feuille permet d’organiser un type de données spécifique.

Exemple :
- Feuille 1 : Dépenses
- Feuille 2 : Revenus

---

### 1.3 Les colonnes
Les colonnes sont des ensembles de cellules disposées verticalement.  
Elles sont identifiées par des lettres (A, B, C, ...).

---

### 1.4 Les lignes
Les lignes sont des ensembles de cellules disposées horizontalement.  
Elles sont identifiées par des numéros (1, 2, 3, ...).

---

### 1.5 Les cellules
Une cellule correspond à l’intersection d’une ligne et d’une colonne.  
Elle est utilisée pour saisir des données.

Notation :
- A1 : colonne A, ligne 1
- B3 : colonne B, ligne 3

La cellule active est celle qui est actuellement sélectionnée.

---

## 2. Description de l’interface Excel

### 2.1 Le ruban
Le ruban est la zone principale de commande située en haut de la fenêtre.  
Il regroupe l’ensemble des outils nécessaires à l’utilisation d’Excel.

---

### 2.2 Les onglets
Les onglets permettent d’organiser les fonctionnalités du ruban.

Principaux onglets :
- Accueil : saisie et mise en forme
- Insertion : ajout d’objets (tableaux, graphiques)
- Mise en page : gestion de l’impression
- Formules : fonctions et calculs
- Données : tri et filtrage

Chaque onglet contient des groupes d’outils spécialisés.

---

### 2.3 La barre de formule
La barre de formule permet :
- d’afficher le contenu d’une cellule
- de modifier ce contenu

Elle est essentielle pour visualiser les formules et les valeurs.

---

## 3. Navigation dans Excel

### 3.1 Navigation avec la souris
- Cliquer sur une cellule pour la sélectionner
- Utiliser la molette pour se déplacer verticalement
- Utiliser les barres de défilement pour naviguer

---

### 3.2 Navigation avec le clavier
- Flèches directionnelles : déplacement cellule par cellule
- Entrée : déplacement vers le bas
- Tabulation : déplacement vers la droite

---

### 3.3 Navigation rapide
- Ctrl + flèche bas : atteindre la dernière cellule remplie d’une colonne
- Ctrl + flèche haut : revenir au début des données
- Ctrl + flèche droite : atteindre la fin d’une ligne
- Ctrl + flèche gauche : revenir au début de la ligne

Ces raccourcis sont essentiels pour travailler sur de grands volumes de données.

---

## 4. Sélection de cellules

### 4.1 Sélection simple
Cliquer sur une cellule

---

### 4.2 Sélection d’une plage continue
- Cliquer et faire glisser la souris
- Ou utiliser Shift + flèches directionnelles

---

### 4.3 Sélection de plusieurs zones
Maintenir la touche Ctrl puis cliquer sur différentes cellules ou plages

---

## 5. Cas pratique

### 5.1 Objectif
Créer un tableau simple représentant une liste de courses.

---

### 5.2 Étapes

1. Ouvrir Microsoft Excel et créer un nouveau classeur

2. Saisir les en-têtes :
   - Cellule A1 : Produit
   - Cellule B1 : Prix

3. Saisir les données :
   - A2 : Riz       | B2 : 5000
   - A3 : Huile     | B3 : 3000
   - A4 : Pain      | B4 : 2000
   - A5 : Sucre     | B5 : 1500

4. Sélectionner la plage A1:B5

5. Tester les déplacements :
   - Utiliser les flèches directionnelles
   - Utiliser Ctrl + flèche bas pour atteindre rapidement la dernière ligne

6. Cliquer sur différentes cellules et observer le contenu dans la barre de formule

---

### 5.3 Résultat attendu
Un tableau structuré contenant une liste de produits et leurs prix, organisé en lignes et colonnes.

---
