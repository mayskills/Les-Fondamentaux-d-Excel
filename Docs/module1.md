# Module 1 – Interface et Navigation

## Objectif pédagogique
À la fin de ce module, tu dois être capable de :
- Comprendre comment Excel est organisé
- Te repérer facilement dans un fichier Excel
- Naviguer rapidement sans te perdre
- Sélectionner les bonnes cellules au bon moment

L’idée ici n’est pas de mémoriser, mais de te sentir à l’aise dans Excel.

---

## 1. Comprendre la structure d’Excel

Avant de cliquer partout, il faut comprendre ce que tu as devant toi.

### 1.1 Le classeur
Un classeur, c’est simplement ton fichier Excel.

<p align="center">
  <img src="../img/Le classeur.jpg" width="500">
</p>

Imagine un cahier :
- Le fichier Excel = le cahier
- À l’intérieur, tu peux avoir plusieurs pages

Exemple concret :
- Une page pour les dépenses
- Une page pour les revenus
- Une page pour le budget global

---

### 1.2 Les feuilles
Une feuille est une page à l’intérieur du classeur.

Chaque feuille sert à organiser un type de données.

Exemple simple :
- Feuille 1 : Budget mensuel
- Feuille 2 : Liste de courses
- Feuille 3 : Suivi des ventes

👉 En pratique, on travaille souvent avec plusieurs feuilles dans un même fichier.

---

### 1.3 Les colonnes
Les colonnes sont les lignes verticales dans Excel.

Elles sont identifiées par des lettres :
A, B, C, D, etc.

Exemple :
- Colonne A : Produits
- Colonne B : Prix
- Colonne C : Quantité

---

### 1.4 Les lignes
Les lignes sont les lignes horizontales.

Elles sont identifiées par des chiffres :
1, 2, 3, 4, etc.

Exemple :
- Ligne 1 : en-têtes
- Ligne 2 : première donnée
- Ligne 3 : deuxième donnée

---

### 1.5 Les cellules
Une cellule est le point d’intersection entre une ligne et une colonne.

Exemple :
- A1 = colonne A + ligne 1
- B3 = colonne B + ligne 3

👉 C’est dans les cellules que tu écris tes données.

---

## 2. Comprendre l’interface Excel

Quand tu ouvres Excel, tu vois plusieurs zones importantes.

---

### 2.1 Le ruban
Le ruban est la grande barre en haut de l’écran.

C’est là que tu trouves tous les outils :
- écrire
- formater
- insérer des graphiques
- faire des calculs

👉 C’est ton “centre de commande”

---

### 2.2 Les onglets
Les onglets sont les catégories du ruban.

Les plus importants :
- Accueil : écrire et mettre en forme
- Insertion : tableaux, graphiques
- Mise en page : impression
- Formules : calculs
- Données : tri et filtrage

👉 Chaque onglet a un rôle précis.

---

### 2.3 La barre de formule
La barre de formule est située au-dessus des cellules.

Elle te permet de :
- voir ce que contient une cellule
- modifier une donnée
- voir les formules utilisées

👉 C’est très important pour comprendre ce que fait Excel derrière.

---

## 3. Se déplacer dans Excel

### 3.1 Avec la souris
C’est la méthode la plus simple :
- cliquer sur une cellule
- faire défiler avec la molette

👉 Mais ce n’est pas la plus rapide.

---

### 3.2 Avec le clavier
C’est la méthode des personnes rapides :

- Flèches : bouger cellule par cellule
- Entrée : descendre
- Tabulation : aller à droite

👉 Plus tu utilises le clavier, plus tu gagnes du temps.

---

### 3.3 Navigation rapide (très important)

Ces raccourcis sont utilisés tous les jours en entreprise :

- Ctrl + flèche bas : aller à la dernière donnée d’une colonne
- Ctrl + flèche haut : revenir en haut
- Ctrl + flèche droite : aller à droite jusqu’à la fin des données
- Ctrl + flèche gauche : revenir au début

👉 Exemple réel :
Si tu as 1000 lignes, tu n’as pas besoin de descendre une par une.

---

## 4. Sélection des cellules

### 4.1 Sélection simple
- Cliquer sur une cellule

---

### 4.2 Sélection d’une zone
- Cliquer et glisser avec la souris
- Ou utiliser Shift + flèches

---

### 4.3 Sélection de plusieurs zones
- Maintenir Ctrl
- Cliquer sur différentes cellules

---

## 5. Cas pratique

### 5.1 Situation
Tu veux créer une petite liste de courses comme un tableau Excel réel.

---

### 5.2 Étapes

1. Ouvre Excel et crée un nouveau fichier

2. Dans la ligne 1, écris :
- A1 : Produit
- B1 : Prix

3. Remplis les données :
- A2 : Riz       | B2 : 5000
- A3 : Huile     | B3 : 3000
- A4 : Pain      | B4 : 2000
- A5 : Sucre     | B5 : 1500

4. Clique sur A1:B5 et observe que tout est bien structuré

5. Teste les déplacements :
- flèches du clavier
- Ctrl + flèche bas
- Ctrl + flèche haut

6. Clique sur plusieurs cellules et regarde la barre de formule

---

### 5.3 Ce que tu dois observer
- Chaque donnée est dans une cellule précise
- Tu peux te déplacer très rapidement
- La structure reste claire et lisible

---

## 6. Question de contrôle

Sans regarder le cours, réponds :

Que se passe-t-il quand tu appuies sur Ctrl + flèche bas dans une colonne remplie de données ?

---

## 7. Conseils importants

- Ne te presse pas au début
- Répète les manipulations plusieurs fois
- Apprends à utiliser le clavier progressivement
- Travaille toujours avec des exemples concrets (budget, courses, dépenses)

Si tu comprends bien ce module, tu as déjà fait un grand pas.

Excel n’est pas compliqué, mais il faut apprendre à se repérer dedans comme dans une maison.

Une fois que tu es à l’aise avec l’interface, tout le reste devient beaucoup plus simple.
